![](C:\Users\30546\AppData\Roaming\Typora\typora-user-images\image-20250330165323038.png)![图片.png](images/img_17575_065.png)

解压后是raw的镜像

我们可以用volatility来做

volatility有vol2 和vol3 两个版本 分别是用python2 python3来运行

在进行内存取证的时候将vol2和vol3进行结合使用，vol2能弥补3插件不足，3能弥补2的速度慢，但在做题的过程中，一般都是以vol2为主

```
python2 vol.py -f shuzi.raw imageinfo
```

通过imageinfo来确认这个镜像对应的win版本

![](C:\Users\30546\AppData\Roaming\Typora\typora-user-images\image-20250330165827257.png)![图片.png](images/img_17575_067.png)

可以知道是Win7SP1X64(其实大部分也是这个)

```
python2 vol.py -f shuzi.raw --profile=Win7SP1x64 pslist
```

之后来看进程![](C:\Users\30546\AppData\Roaming\Typora\typora-user-images\image-20250330165929089.png)

![图片.png](images/img_17575_069.png)

存在TrueCrypt.exe PID是628

TrueCrypt是一个文件加密的exe，我们的log文件就是被这个exe所加密的

```
python2 vol.py -f shuzi.raw --profile=Win7SP1x64 memdump -p 628 -D ./    
```

利用memdump插件 将这个TrueCrypt.exe在内存中的数据导出来
