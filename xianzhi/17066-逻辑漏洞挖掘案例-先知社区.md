# 逻辑漏洞挖掘案例-先知社区

> **来源**: https://xz.aliyun.com/news/17066  
> **文章ID**: 17066

---

开局一个登录框，发现有忘记密码功能

忘记密码功能可能存在的漏洞：

```
用户名遍历
（绕过分布校验）任意用户密码重置
（重置密码链接可猜解）任意用户密码重置
（未校验手机号码与用户的一致性）任意用户密码重置
（修改用户id/用户名）任意用户密码重置
```

# 案例一

![](images/20250307143355-2498188a-fb1e-1.png)

这里有3种找回密码方式，先看手机号码，随便输入一个电话号码（信息收集）和验证码，进行抓包

![](images/20250307143356-258aa131-fb1e-1.png)

看响应和请求，发现可能存在逻辑漏洞

![](images/20250307143358-263f5ac4-fb1e-1.png)

![](images/20250307143359-272b2964-fb1e-1.png)把false改为true,看看能不能绕过去![](images/20250307143400-27a2c33a-fb1e-1.png)

到这里就发现并不需要验证码就可以绕过重置密码

![](images/20250307143401-28431e2d-fb1e-1.png)
